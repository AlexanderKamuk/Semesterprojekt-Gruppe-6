This is a git repository containing all software used to complete the two courses for 1. semester project in mobile robotics.
As welle as figures used in the rapport and pictures of the actual robots themselves.

The repository is organised with a folder containing software for the track driving course, another folder containing software for the logistics course, a folder with figures, and a folder with pictures of the robots.

The track driving folder contains a main loop and relevant software to make the robot follow the marked track.

The logistics folder contains software for track driving, detection, electromagnet activation, and actuator control.

The figures folder contains all figures used in the rapport.

The pictures folder contains actual pictures of the two robots.
